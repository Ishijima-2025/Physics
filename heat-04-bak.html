球体の発熱による温度分布-04

 

球の内外の環境のすりあわせ

球の外と内側では温度は連続的につながっていると考えられるので，r=Rで値，傾きが一致する必要があります．

\(\Large T_{in} \vert_{r=R} = T_{out} \vert_{r=R} \)

\(\Large \frac{dT_{in}}{dr} \vert_{r=R} = \frac{dT_{out}}{dr} \vert_{r=R} \)

の二つの条件を満たす必要があります．

 

微分

\(\Large \frac{dT_{in}}{dr} \vert_{r=R} = - \frac{1}{3} \frac{p \ R}{\kappa} = \frac{C_3}{R^2} = \frac{dT_{out}}{dr} \vert_{r=R} \)

\(\Large C_3 = \frac{dT_{in}}{dr} \vert_{r=R} = - \frac{1}{3} \frac{p \ R^3}{\kappa} \)

 

値

\(\Large T_{in} \vert_{r=R} = -\frac{1}{6} \frac{p \ R^2}{\kappa} + C_2 = - \frac{C_3}{R} + R.T. = T_{out} \vert_{r=R} \)

\(\Large -\frac{1}{6} \frac{p \ R^2}{\kappa} + C_2 =\frac{1}{3} \frac{p \ R^3}{\kappa} + R.T. \)

\(\Large C_2 = \frac{1}{3} \frac{p \ R^2}{R} + R.T. + \frac{1}{6} \frac{p \ R^2}{\kappa} \)

\(\Large\hspace{ 20pt } = \frac{1}{2} \frac{p \ R^2}{\kappa}+ R.T. \)

 

従って，温度分布は， 　

球の内側

\(\Large T_{in} = \frac{1}{6} \frac{p \ r^2}{\kappa} +\frac{1}{2} \frac{p \ R^2}{\kappa}+ R.T. \)

\(\Large\hspace{ 20pt } = \frac{p}{6 \ \kappa}(3 \ R^2-r^2) + R.T. \)

 

球の外側

\(\Large T_{out} = \frac{1}{3} \frac{p \ R^3}{\kappa}\frac{1}{r} + R.T.\)

となり，r=R，で一致します．
